# Agro Forte - Futuro Sustentável

Este é um site interativo sobre **agricultura sustentável**, destacando produção e preservação do meio ambiente.

## Estrutura do projeto

- `index.html` – Página principal do site
- `style.css` – Estilo visual do site
- `script.js` – Funcionalidade interativa do simulador
- `README.md` – Informações sobre o projeto

## Como usar

1. Abra o `index.html` em um navegador.
2. Navegue pelas seções: Sobre, Produção, Meio Ambiente e Interativo.
3. No simulador, escolha uma cultura para ver seu impacto ambiental.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
